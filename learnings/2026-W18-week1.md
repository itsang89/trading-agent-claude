## New Rules Extracted (3)

### Sizing and Risk

**Rule 1** [W1|MEDIUM] — Overnight gap risk: proactive trim on RS session-1 negative.
- When a held position triggers its FIRST RS_spread < −1% session, consider trimming to a lower conviction tier rather than holding through the 2-session exit confirmation. This reduces overnight gap exposure.
- Context: META closed ~$669 on 4/29, opened at $609 on 4/30 — $60 gap below the $621.86 hard stop. Resulted in −10.62% realized loss vs intended −8% cap.

**Rule 2** [W1|MEDIUM] — Trailing stop deactivation on adds.
- When adding to a position with an ACTIVE trailing stop, document that the add will deactivate the trailing stop (avg_entry rises → threshold rises above high_close). State explicitly whether this is acceptable given current size and signals.
- Context: GOOGL add on 5/1 raised avg_entry from $346.55 to $366.98, deactivating an active trailing stop on a 13% position (new threshold $403.68 vs high_close $385.79).

### Behavioral Failure Modes

**Rule 3** [W1|HIGH] — Model deviation logging.
- If a routine runs under a non-scheduled model, document the actual model in the journal header AND note-for-operator immediately. Do not suppress.
- Context: 5 sessions used "opencode/hy3-preview-free" instead of claude-sonnet-4-6; only discoverable via journal headers.
